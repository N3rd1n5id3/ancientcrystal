# Pokémon Ancient Crystal

## Features

* Restored the GS Ball Celebi Event
* Colored trainer card badges and third Trainer Card page for Kanto badges.
* Clock reset (D-Pad Down + B buttons on title screen)
* Removed the artificial save delay
* Kurt finishes Apricorn Pokeballs instantly
* Smashing rocks has a chance to contain items
* Unique colors for each thrown Poké Ball
* Added Helix Fossil, Domo Fossil, Old Amber, Fossil Egg and restored Pewter museum.
